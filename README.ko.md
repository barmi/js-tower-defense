# JS Tower Defense

간단한 브라우저 기반 그리드 타워 디펜스 게임입니다.  
(바닐라 HTML/CSS/JS, 프레임워크 없음)

영문 버전: [README.md](./README.md)

## 실행 방법

1. 브라우저에서 `./index.html` 파일을 엽니다.
2. 빈 타일에 타워를 설치합니다.
3. `Start`를 눌러 첫 웨이브를 시작합니다.

## 현재 기능

### 기본 게임플레이
- 멀티 턴 경로를 사용하는 그리드 맵.
- 웨이브 진행에 따라 난이도가 상승하는 적 스폰.
- 골드/라이프/웨이브 기반 진행 루프.
- 단일 버튼으로 일시정지/재개 지원.

### 타워 시스템
- 타워 3종: `Rifle`, `Rapid`, `Sniper`.
- 타입별 스탯: 비용, 공격력, 공격 속도, 사거리, 투사체 속도.
- 빌드 선택 패널.
- 설치 전 사거리 프리뷰(별도 색상).
- 발사 시 목표 방향으로 포탑 조준.

### 타워 선택 및 업그레이드
- 타워 클릭 시 선택.
- 선택 타워 하이라이트 및 사거리 표시.
- 업그레이드 패널에서 현재 스탯/다음 업그레이드 비용 확인.
- 타워 판매 시 일부 골드 환급.
- 레벨에 따른 타워 외형 차이.

### 적 시스템
- 적 5종: `Scout`, `Grunt`, `Raider`, `Tank`, `Juggernaut`.
- 원형 대신 타입별 단순 형상 사용.
- 타입별 스탯 분리(HP/속도/보상/유출 피해).
- 웨이브 번호에 따라 적 조합 변화.

### 웨이브 흐름
- `Start` 시 즉시 웨이브 시작.
- 웨이브 클리어 후 짧은 대기 뒤 자동 다음 웨이브 시작.
- `Skip Wait`로 즉시 다음 웨이브 시작 가능.

### UI / 언어
- 게임 내 UI 문구는 간결한 영어 사용.
- 빌드 목록의 타워 이름 왼쪽에 타입 아이콘 표시.

## 주요 파일

- `./index.html`: 레이아웃 및 UI 구조
- `./style.css`: 스타일 정의
- `./app.js`: 게임 로직 전체
